<h1>Isi Repositori</h1>
Repositori ini berisi kode Python yang dirancang untuk mengambil (scraping) data ulasan dari Google Maps dan Google Play Store. Menggunakan library Selenium. Dalam Repositori ini terdapat fitur scroll otomatis yang dapat digunakan untuk mensimulasikan pengguna dalam proses pengambilan data


Repositori ini dibuat untuk tujuan pembelajaran. Gunakanlah secara bijak.

<i>Note: Jika scroll otomatis tidak berjalan. Dapat dibantu dengan cara scroll secara manual terlebih dahulu</i>
